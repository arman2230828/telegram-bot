OBJECTIVE

Create a professional Telegram cloud storage bot similar to FreeStorageBot where users can upload APKs, ZIP files, PDFs, Videos, Audios, Images, Documents, and other Telegram-supported files.

Files must be stored inside a private Telegram storage channel while only metadata and file IDs are stored in MySQL.

The bot should generate permanent shareable download links and support thousands of users efficiently.

USER FLOW

1. User starts bot.
2. Force Join verification runs.
3. User joins required channel.
4. Home menu appears.
5. User uploads file.
6. Bot forwards file to private storage channel.
7. Bot saves file metadata in MySQL.
8. Bot generates unique file code.
9. Bot creates permanent download link.
10. User receives shareable link.

DOWNLOAD FLOW

1. User opens deep link.
2. Bot checks force join.
3. Bot retrieves file from database.
4. Bot increases download counter.
5. Bot sends file using Telegram file_id.
6. User receives file instantly.

FORCE JOIN SYSTEM (ADVANCED)

The bot must support dynamic force join channels managed entirely from the admin panel.

Rules:

1. If no channels are added in the admin panel:
   
   - Force join must be disabled automatically.
   - User should directly see the Home Menu.
   - User can immediately upload files.

2. If one or more channels are added:
   
   - Force join must be enabled automatically.
   - When the user presses /start, show a force join menu similar to FreeStorageBot.

Message:

🚫 Access Denied

You must join all required channels to use this bot.

After joining, click ✅ I Joined.

Buttons:

📢 Join Channel 1
📢 Join Channel 2
📢 Join Channel 3
📢 Join Private
✅ I Joined

3. Verification Logic

When user clicks "✅ I Joined":

- Check membership in every required channel.
- If user joined all channels:

Show:

✅ Verification successful!

You can now use the bot.

Then automatically delete or edit the previous force join message.

Immediately display the Home Menu.

- If user has not joined all channels:

Show:

❌ Verification failed!

You must join all required channels before using this bot.

Keep the force join menu visible.

4. Message Editing

The force join message must not create new messages repeatedly.

Use message.edit_text() whenever possible.

Old menus should be automatically removed or edited.

Only one active menu should remain visible.

5. Admin Panel Controls

Admin must be able to:

- Add Force Join Channel
- Remove Force Join Channel
- Enable Force Join
- Disable Force Join
- Edit Force Join Message Text
- Edit Button Text
- Reorder Channels

6. Customizable Text

Admin panel must allow editing:

- Access Denied message
- Verification Success message
- Verification Failed message
- Join Button text
- Home Menu welcome message

No code changes should be required to edit these texts.

All text must be stored in the database and editable from the admin panel.
